# Recommended Learning Path (Before Starting Phase 0)

These are free and take a few hours total. Do them in order.

## 1. Kotlin Basics (30 min)
- Kotlin Playground: https://play.kotlinlang.org
- Focus on: variables, functions, classes, null safety, lambdas

## 2. Jetpack Compose Basics (2 hours)
- Google Codelab: https://developer.android.com/codelabs/jetpack-compose-basics
- Focus on: composable functions, state, layouts, modifiers

## 3. Android Audio Concepts (2 hours)
- Google Codelab "Making Waves": https://developer.android.com/codelabs/making-waves-1-synth
- Focus on: how audio works on Android, buffers, sample rates

## 4. MWEngine Overview (30 min)
- Read the "Getting Started" wiki page on the MWEngine GitHub
- Read "Creating a Simple Drum Machine" example
- Link: https://github.com/igorski/MWEngine/wiki

## Total warm-up time: ~5 hours
